STYLING NOTES
================================================================================

Color Coding (consistent with existing):
  - Singleton:  #a6e3a1 (green)   - var(--hex-devtools-singleton)
  - Scoped:     #89b4fa (blue)    - var(--hex-devtools-scoped)
  - Request:    #fab387 (orange)  - var(--hex-devtools-request)

New Status Indicators:
  - Resolved:     Filled circle (#a6e3a1 green)
  - Not Resolved: Empty circle outline (muted text color)

Font:
  - Same monospace font family: var(--hex-devtools-font-mono)
  - Service names: 13px, fontWeight 500
  - Labels/badges: 10px, fontWeight 600, uppercase

Background Colors:
  - Container bg: var(--hex-devtools-bg)           #1e1e2e
  - Section bg:   var(--hex-devtools-bg-secondary) #2a2a3e
  - Hover bg:     var(--hex-devtools-bg-hover)     #3a3a4e
