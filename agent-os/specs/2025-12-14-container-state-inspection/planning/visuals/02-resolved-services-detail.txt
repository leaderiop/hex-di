FILTER STATES
================================================================================

All Filters (toggle buttons):

  +-----------------------------------------------------------------------+
  |  [All] [Singleton] [Scoped] [Request] | [Resolved] [Pending]          |
  +-----------------------------------------------------------------------+

  - First group: Lifetime filters (mutually exclusive with "All")
  - Second group: Status filters (can combine with lifetime)

Active filter styling:
  - Background: var(--hex-devtools-accent) #89b4fa
  - Color: #1e1e2e (dark text on light bg)

Inactive filter styling:
  - Background: transparent
  - Border: 1px solid var(--hex-devtools-border)
  - Color: var(--hex-devtools-text-muted)

================================================================================
ACCESSIBILITY CONSIDERATIONS
================================================================================

1. Resolution status uses both color AND shape:
   - Resolved: Filled circle (shape indicates "full/complete")
   - Not resolved: Empty circle (shape indicates "empty/waiting")

2. All interactive elements have:
   - role="button" for headers
   - aria-expanded for collapsible sections
   - tabIndex={0} for keyboard navigation

3. Filter buttons are toggle buttons with aria-pressed state

4. Status text accompanies visual indicators for screen readers:
   - "Logger - Singleton - Resolved"
   - "UserRepository - Scoped - Not Resolved"
