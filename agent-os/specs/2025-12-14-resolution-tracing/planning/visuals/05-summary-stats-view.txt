OVERVIEW CARDS DETAIL:
--------------------------------------------------------------------------------

Single Card Structure:

+-------------------+
|  LABEL            |  <-- Text muted, uppercase, 10px
|                   |
|      VALUE        |  <-- Large text, accent color, 24px
|                   |
|    (subtext)      |  <-- Text muted, 11px
+-------------------+

Card Styles:
  - background: var(--hex-devtools-bg-secondary)
  - border: 1px solid var(--hex-devtools-border)
  - border-radius: 6px
  - padding: 16px
  - min-width: 100px
  - text-align: center

Cards Grid:
  - display: grid
  - grid-template-columns: repeat(auto-fit, minmax(100px, 1fr))
  - gap: 12px

Conditional Styling:
  - SLOW COUNT card turns red background when count > 0
  - CACHE HIT RATE shows green when > 50%, yellow when 25-50%, red when < 25%

RESPONSIVE LAYOUT:
--------------------------------------------------------------------------------

Wide (> 600px):
  - Cards in 4-column grid
  - Charts side by side where possible

Medium (400-600px):
  - Cards in 2-column grid
  - Charts stacked vertically

Narrow (< 400px):
  - Cards in single column
  - All content stacked
  - Simplified bar charts
